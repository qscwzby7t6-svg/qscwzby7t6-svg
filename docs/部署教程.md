## 1. Windows 本地部署

### 1.1 环境准备

#### 1.1.1 安装 Python 3.10+

1. 访问 [Python官网](https://www.python.org/downloads/windows/)
2. 下载 Python 3.10 或更高版本（建议 3.10.11）
3. **安装时勾选** "Add Python to PATH"
4. 安装完成后验证：
   ```cmd
   python --version
   pip --version
   ```

#### 1.1.2 安装 Git

1. 访问 [Git官网](https://git-scm.com/download/win)
2. 下载并安装 Git for Windows
3. 验证安装：
   ```cmd
   git --version
   ```

### 1.2 克隆项目

```cmd
git clone https://github.com/your-username/ai-song-creator.git
cd ai-song-creator
```

### 1.3 创建虚拟环境（推荐）

```cmd
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate.bat

# 查看虚拟环境是否激活（命令行前会显示 (venv)）
```

### 1.4 安装依赖

```cmd
pip install -r requirements.txt
```

> **注意**: 如果安装 torch 失败，请使用以下命令：
> ```cmd
> pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
> ```

### 1.5 运行应用

```cmd
# 开发模式
streamlit run app.py

# 生产模式（指定端口）
streamlit run app.py --server.port=8501
```

### 1.6 访问应用

打开浏览器访问：`http://localhost:8501`

## 2. 云服务器部署

### 2.1 环境准备（以 Ubuntu 22.04 为例）

#### 2.1.1 更新系统

```bash
sudo apt update && sudo apt upgrade -y
```

#### 2.1.2 安装依赖

```bash
# 安装 Python 和 Git
sudo apt install python3 python3-pip git -y

# 安装音频依赖
sudo apt install libsndfile1 ffmpeg -y
```

#### 2.1.3 安装虚拟环境工具

```bash
sudo apt install python3-venv -y
```

### 2.2 克隆项目

```bash
git clone https://github.com/your-username/ai-song-creator.git
cd ai-song-creator
```

### 2.3 创建虚拟环境

```bash
python3 -m venv venv
source venv/bin/activate
```

### 2.4 安装依赖

```bash
pip install -r requirements.txt
```

### 2.5 使用 systemd 配置服务（推荐）

创建服务文件 `/etc/systemd/system/ai-song-creator.service`：

```ini
[Unit]
Description=AI Song Creator Service
After=network.target

[Service]
User=www-data
Group=www-data
WorkingDirectory=/path/to/ai-song-creator
ExecStart=/path/to/ai-song-creator/venv/bin/streamlit run app.py --server.port=8501 --server.headless=true
Restart=always
Environment=PATH=/path/to/ai-song-creator/venv/bin

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
# 重新加载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start ai-song-creator

# 设置开机自启
sudo systemctl enable ai-song-creator

# 查看服务状态
sudo systemctl status ai-song-creator
```

### 2.6 配置 Nginx 反向代理（可选）

安装 Nginx：

```bash
sudo apt install nginx -y
```

创建配置文件 `/etc/nginx/sites-available/ai-song-creator`：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:8501;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

启用配置：

```bash
sudo ln -s /etc/nginx/sites-available/ai-song-creator /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

### 2.7 配置 HTTPS（推荐）

使用 Certbot 配置 Let's Encrypt：

```bash
sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d your-domain.com
```

## 3. AI Learning 部署

### 3.1 环境准备

#### 3.1.1 安装 Anaconda

```bash
# 下载 Anaconda
wget https://repo.anaconda.com/archive/Anaconda3-2023.09-0-Linux-x86_64.sh

# 安装 Anaconda
bash Anaconda3-2023.09-0-Linux-x86_64.sh

# 激活环境
source ~/.bashrc
```

#### 3.1.2 创建 Conda 环境

```bash
conda create -n ai-song-creator python=3.10 -y
conda activate ai-song-creator
```

### 3.2 安装依赖

```bash
# 安装 PyTorch (GPU版本)
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia

# 安装其他依赖
pip install -r requirements.txt
```

### 3.3 运行应用

```bash
streamlit run app.py --server.port=8501 --server.headless=true
```

### 3.4 Jupyter Notebook 集成

```bash
# 安装 Jupyter
conda install jupyter -y

# 创建 Notebook 目录
mkdir notebooks
cd notebooks

# 创建示例 Notebook
jupyter notebook
```

## 5. 常见问题

### Q1: Streamlit 启动失败

**问题**：`ModuleNotFoundError: No module named 'streamlit'`

**解决方案**：
```bash
pip install streamlit
```

### Q2: Librosa 音频处理失败

**问题**：`libsndfile not found`

**解决方案**：
```bash
# Ubuntu/Debian
sudo apt install libsndfile1

# macOS
brew install libsndfile

# Windows
# 确保安装了 soundfile
pip install soundfile
```

### Q3: PyTorch 安装失败

**问题**：torch 安装失败或版本不兼容

**解决方案**：
```bash
# CPU版本
pip install torch==2.0.0

# GPU版本（根据CUDA版本）
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Q4: 端口被占用

**问题**：`Address already in use`

**解决方案**：
```bash
# 查看端口占用
lsof -i :8501

# 杀死占用进程
kill -9 <PID>

# 使用其他端口
streamlit run app.py --server.port=8502
```

### Q5: 数据目录权限问题

**问题**：无法写入数据目录

**解决方案**：
```bash
chown -R www-data:www-data data/
chmod -R 755 data/
```
