## 1. Windows 本地部署

### 1.1 环境准备

#### 1.1.1 安装 Python 3.10+

1. 访问 [Python官网](https://www.python.org/downloads/windows/)
2. 下载 Python 3.10 或更高版本（建议 3.10.11）
3. **安装时勾选** "Add Python to PATH"
4. 安装完成后验证：
   ```cmd
   python --version
   pip --version
   ```

#### 1.1.2 安装 Git

1. 访问 [Git官网](https://git-scm.com/download/win)
2. 下载并安装 Git for Windows
3. 验证安装：
   ```cmd
   git --version
   ```

### 1.2 克隆项目

```cmd
git clone https://github.com/qscwzby7t6-svg/qscwzby7t6-svg.git
cd qscwzby7t6-svg
```

### 1.3 创建虚拟环境（推荐）

```cmd
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate.bat

# 查看虚拟环境是否激活（命令行前会显示 (venv)）
```

### 1.4 安装依赖

```cmd
pip install -r requirements.txt
```

> **注意**: 如果安装 torch 失败，请使用以下命令：
> ```cmd
> pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
> ```

### 1.5 运行应用

```cmd
# 开发模式
streamlit run app.py

# 生产模式（指定端口）
streamlit run app.py --server.port=8501
```

### 1.6 访问应用

打开浏览器访问：`http://localhost:8501`

## 2. 云服务器部署

### 2.1 环境准备（以 Ubuntu 22.04 为例）

#### 2.1.1 更新系统

```bash
sudo apt update && sudo apt upgrade -y
```

#### 2.1.2 安装依赖

```bash
# 安装 Python 和 Git
sudo apt install python3 python3-pip git -y

# 安装音频依赖
sudo apt install libsndfile1 ffmpeg -y
```

#### 2.1.3 安装虚拟环境工具

```bash
sudo apt install python3-venv -y
```

### 2.2 克隆项目

```bash
git clone https://github.com/qscwzby7t6-svg/qscwzby7t6-svg.git
cd qscwzby7t6-svg
```

### 2.3 创建虚拟环境

```bash
python3 -m venv venv
source venv/bin/activate
```

### 2.4 安装依赖

```bash
pip install -r requirements.txt
```

### 2.5 使用 systemd 配置服务（推荐）

创建服务文件 `/etc/systemd/system/ai-song-creator.service`：

```ini
[Unit]
Description=AI Song Creator Service
After=network.target

[Service]
User=www-data
Group=www-data
WorkingDirectory=/path/to/qscwzby7t6-svg
ExecStart=/path/to/qscwzby7t6-svg/venv/bin/streamlit run app.py --server.port=8501 --server.headless=true
Restart=always
Environment=PATH=/path/to/qscwzby7t6-svg/venv/bin

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
# 重新加载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start ai-song-creator

# 设置开机自启
sudo systemctl enable ai-song-creator

# 查看服务状态
sudo systemctl status ai-song-creator
```

### 2.6 配置 Nginx 反向代理（可选）

安装 Nginx：

```bash
sudo apt install nginx -y
```

创建配置文件 `/etc/nginx/sites-available/ai-song-creator`：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:8501;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

启用配置：

```bash
sudo ln -s /etc/nginx/sites-available/ai-song-creator /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

### 2.7 配置 HTTPS（推荐）

使用 Certbot 配置 Let's Encrypt：

```bash
sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d your-domain.com
```

## 3. AI Learning 部署

### 3.1 环境准备

#### 3.1.1 检查系统要求

AI Learning 通常基于 Android/Linux 环境，需要确保：
- Android 7.0+ 或 Linux 环境
- 至少 4GB 可用存储空间
- 稳定的网络连接

#### 3.1.2 安装 Termux（Android 用户）

1. 从 F-Droid 下载并安装 Termux
2. 更新 Termux 包：
   ```bash
   pkg update && pkg upgrade -y
   ```

#### 3.1.3 安装必要工具

```bash
# 安装基础工具
pkg install git python python-pip wget -y

# 或 apt（Linux环境）
apt update && apt install git python3 python3-pip wget -y
```

### 3.2 安装 Miniconda（推荐）

```bash
# 下载 Miniconda
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-aarch64.sh

# 安装 Miniconda
bash Miniconda3-latest-Linux-aarch64.sh -b -p $HOME/miniconda3

# 初始化 Conda
$HOME/miniconda3/bin/conda init bash

# 重新加载配置
source ~/.bashrc
```

> **注意**: 如果是 x86_64 架构，请使用 `Miniconda3-latest-Linux-x86_64.sh`

### 3.3 克隆项目

```bash
# 创建项目目录
mkdir -p ~/projects
cd ~/projects

# 克隆项目
git clone https://github.com/qscwzby7t6-svg/qscwzby7t6-svg.git
cd qscwzby7t6-svg
```

### 3.4 创建 Conda 环境

```bash
# 创建环境
conda create -n ai-song-creator python=3.10 -y

# 激活环境
conda activate ai-song-creator

# 验证 Python 版本
python --version  # 应显示 Python 3.10.x
```

### 3.5 安装依赖

#### 3.5.1 安装系统依赖（Termux）

```bash
# 安装音频处理依赖
pkg install libsndfile ffmpeg -y

# 安装编译工具（可能需要）
pkg install clang pkg-config -y
```

#### 3.5.2 安装 Python 依赖

```bash
# 升级 pip
pip install --upgrade pip

# 安装 numpy 和 scipy（先安装避免兼容问题）
pip install numpy==1.24.0 scipy==1.11.0

# 安装 PyTorch（CPU版本，AI Learning通常无GPU）
pip install torch==2.0.0 torchvision==0.15.0 torchaudio==2.0.0 --index-url https://download.pytorch.org/whl/cpu

# 安装其他依赖
pip install -r requirements.txt
```

> **注意**: AI Learning 环境通常没有 GPU，请使用 CPU 版本的 PyTorch

### 3.6 创建必要目录

```bash
# 创建数据目录
mkdir -p data/audio
mkdir -p data/db

# 设置权限
chmod -R 755 data/
```

### 3.11 AI Learning 特殊注意事项

#### 3.11.1 存储空间管理

```bash
# 查看存储空间
df -h

# 清理缓存
conda clean --all -y
pip cache purge

# 删除不必要的文件
rm -rf __pycache__ .pytest_cache
```

#### 3.11.2 性能优化

```bash
# 设置环境变量优化性能
export OMP_NUM_THREADS=4
export MKL_NUM_THREADS=4

# 添加到 ~/.bashrc
echo 'export OMP_NUM_THREADS=4' >> ~/.bashrc
echo 'export MKL_NUM_THREADS=4' >> ~/.bashrc
```

#### 3.11.3 常见问题处理

**问题1**: `librosa` 安装失败
```bash
# 解决方案：安装系统依赖后重试
pkg install libsndfile ffmpeg -y
pip install librosa --no-cache-dir
```

**问题2**: `soundfile` 导入错误
```bash
# 解决方案：重新安装 soundfile
pip uninstall soundfile -y
pip install soundfile
```

**问题3**: 内存不足
```bash
# 解决方案：创建 swap 文件
dd if=/dev/zero of=/data/data/com.termux/files/usr/swapfile bs=1M count=2048
mkswap /data/data/com.termux/files/usr/swapfile
swapon /data/data/com.termux/files/usr/swapfile
```

## 5. 常见问题

### Q1: Streamlit 启动失败

**问题**：`ModuleNotFoundError: No module named 'streamlit'`

**解决方案**：
```bash
pip install streamlit
```

### Q2: Librosa 音频处理失败

**问题**：`libsndfile not found`

**解决方案**：
```bash
# Ubuntu/Debian
sudo apt install libsndfile1

# macOS
brew install libsndfile

# Windows
# 确保安装了 soundfile
pip install soundfile

# Termux (AI Learning)
pkg install libsndfile ffmpeg -y
```

### Q3: PyTorch 安装失败

**问题**：torch 安装失败或版本不兼容

**解决方案**：
```bash
# CPU版本（推荐AI Learning使用）
pip install torch==2.0.0 torchvision==0.15.0 torchaudio==2.0.0 --index-url https://download.pytorch.org/whl/cpu

# GPU版本（根据CUDA版本）
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Q4: 端口被占用

**问题**：`Address already in use`

**解决方案**：
```bash
# 查看端口占用
lsof -i :8501

# 杀死占用进程
kill -9 <PID>

# 使用其他端口
streamlit run app.py --server.port=8502
```

### Q5: 数据目录权限问题

**问题**：无法写入数据目录

**解决方案**：
```bash
chown -R www-data:www-data data/
chmod -R 755 data/
```

### Q6: AI Learning 存储空间不足

**问题**：安装过程中提示存储空间不足

**解决方案**：
```bash
# 清理 Conda 缓存
conda clean --all -y

# 清理 pip 缓存
pip cache purge

# 删除不必要的包
conda remove --name ai-song-creator --all -y
conda create -n ai-song-creator python=3.10 -y
```
